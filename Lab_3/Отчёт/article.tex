\subsection{Список моделей}
	В процессе лабораторной работы будем решать задачу бинарной классификации - статус персонажа (жив/мертв).

	Для обучения выберем следующие модели:
	\begin{enumerate}
		\item Наивный байесовский классификатор
		\item Логистическая регрессия
		\item Рандомный лес
		\item Метод опорных векторов
		\item Многослойный перцептрон
	\end{enumerate}
	
	Также модели, поддерживающие гиперпараметры, будут отбираться среди 2-4 набор подготовленных гиперпараметров.

\subsection{Выбор лучшей модели}
	В качестве главной метрики выберем \texttt{ROC-AUC} - метрику для бинарной классификации.
	
	Она показывает долю площади под кривой ROC. 
	
	Кривая ROC отображается на двумерной плоскости, ось X - значение False Positive Rate, ось Y - значение True Positive Rate.
	
	Она показывает зависимость доли истинно положительных ответов (чувствительности) от доли ложно положительных ответов (ошибки I рода)
	
	Значение метрики находится на отрезке \texttt{[0;1]}. Значение метрики в 0.5 описывает случайное предсказание класса; чем ближе значение метрики к единице, тем лучше модель.
	
	По метрике ROC-AUC лучшей моделью является случайный лес с числом деревьев 20.
